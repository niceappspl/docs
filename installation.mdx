💻 CodeWave Studio can be installed on macOS, Windows, or used as a web application. Follow the steps below to get started with your preferred platform.

## macOS Installation

1. 📥 Download the installer from [CodeWave Studio Download Page](#).
2. 🗂️ Open the downloaded file and follow the installation instructions.
3. 🚀 Once installed, open CodeWave Studio from your Applications folder.

## Windows Installation

1. 📥 Download the installer from [CodeWave Studio Download Page](#).
2. 🗂️ Run the downloaded file and follow the installation instructions.
3. 🚀 Once installed, open CodeWave Studio from your Start menu.

## Web Application

1. 🌐 Visit [CodeWave Studio](https://www.codewavestudio.io/).
2. 📝 Register or log in to your account.
3. 🚀 Start using CodeWave Studio directly from your browser.
